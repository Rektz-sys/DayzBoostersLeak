Example Code & Usage
SpawnObject Input Example 
SpawnObject("Land_Village_Store", "5500.123 450.789 7800.456", "0.000000 0.000000 0.000000", 1);
SpawnObject("LootSpot", "5501.234 451.789 7801.567", "0.000000 0.000000 0.000000", 1);
SpawnObject("LootSpot", "5502.345 452.789 7802.678", "0.000000 0.000000 0.000000", 1);
Main building should be first, followed by associated objects
Mapgrouppos XML Example 
<group name="Land_Village_Store" pos="1167.294678 152.374146 6220.293945" rpy="-0.000000 0.000000 -0.492173" a="90.492165" />
Copy from mapgrouppos.xml for the new location
Manual Position Example
X: 1167.294678
Y: 152.374146
Z: 6220.293945
Yaw: -0.492173
Important Notes
  • Main building must have YPR set to 0,0,0 in original SpawnObject calls
  • Objects will maintain their relative positions to the main building
  • Yaw rotation will be applied to the entire group
  • Use either XML input OR manual coordinates, not both

Guide: Using the Spawn Object Mover
Quick Start

This tool helps you move groups of objects in DayZ to new locations while maintaining their relative positions.

1
Select Your Reference Building

In DayZ Editor:

  • Choose a building as your main reference point
  • This will be your "anchor" for positioning other objects
  • Important: Set building rotation (YPR) to 0,0,0
2
Export Your Objects

From DayZ Editor:

  • Export objects as a .C file
  • Reference building should be first in the list
  • Include all objects you want to move together
3
Input Your Data
Option 1: Use XML Data

Paste from mapgrouppos.xml for existing locations

Option 2: Manual Entry

Enter X, Y, Z coordinates manually

4
Generate & Use
  • Click "Generate Output" to create new positions
  • Copy or download the results
  • Use the output in your server configuration
Example Formats
SpawnObject Format:
SpawnObject("Land_Village_store", "4418.120117 341.256989 10938.799805", "0.000000 0.000000 -0.000000", 1);
XML Format:
<group name="Land_Village_store" pos="1167.294678 152.374146 6220.293945" rpy="-0.000000 0.000000 -0.492173" a="90.492165" />
