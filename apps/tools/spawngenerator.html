Spawn Point Generator

Generate spawnpoints using original dayz spawn point xml.

Quick Guide

Click anywhere on the map to add spawn points. Points are automatically saved and displayed in the side panel.

When choosing spawn points, please make sure you have selected the correct map. Incorrect map spawn points will cause water spawns

When testing spawn points, change player save to 0 in db->economy.xml to test faster. Change it back to 1 once you have finished
