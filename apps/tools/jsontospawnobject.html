JSON to INIT.C Converter

Convert your JSON coordinates to SpawnObject format

Convert JSON to SpawnObject
Quick Guide
This tool works with our SpawnObject Mover tool for easy object placement.
Supported JSON Formats:
Standard JSON Format
Uses "Objects" array with name, pos, ypr properties
Unbinarised .dze Format
Uses "EditorObjects" with Type, Position, Orientation
How to Use:
  • Paste your JSON data into the input field
  • Click "Convert to INIT.C" to generate the code
  • Copy the generated code or download as init.c file
  • Import the file into your server configuration
