What does each section mean?

Maximum limit of spawned animals across all zones in map. Doesn't include chickens. The Default is 200. Increasing too much without reducing other events can cause lag.

Distance from player required before item/loot deletion in meters. Default is 100.

The lifetime for dead animals; how long until they despawn in seconds Default is 1200. Tied to 'CleanupAvoidance'.

The lifetime for dead infected; how long until they despawn in seconds Default is 330. Tied to 'CleanupAvoidance'.

The lifetime for dead player; how long until the body despawns in seconds. Default is 3600. Tied to 'CleanupAvoidance'.

The lifetime for things not in types.xml or unrecognized but item is ruined. In seconds default is 45.

How many items can be deleted at once during standard cleanup. Default is 50.

Default lifetime for ruined loot in seconds Default is 330.

The Lifetimes of items near a Raised Flag will be refreshed with this frequency in seconds default is 432000 (5 days).

How long the raised flag will be refreshing items default is 3456000 seconds (40 days).

Allows decay on food. To turn off rotting food, change this AND WorldWetTempUpdate to 0.

Activate economy idle mode on empty server after given time. To keep running set this and IdleModeStartup to 0.

Zero '0' to disable idle mode on server startup; it will still switch on later if IdleModeCountdown is not 0.

How much loot will be spawned on server initial start without storage Default 100.

Maximum loot damage. 0 on min, max is pristine; 0.82 is badly damaged; 1 is ruined.

Minimum loot damage. 0 on min, max is pristine; 1 is ruined.

Allow dispatch containers to receive the loot like car parts and proxies in underground bunker in vanilla.

How far away a player should be from a loot group for loot to spawn inside of it default is 100 meters.

How many attempts are performed during single item respawn the default is 2 it will not attempt to respawn that item again until the next server restart or a new loot refresh cycle.

How many items of ONE type can be spawned at once. The default is 20 meaning a maximum of 20 can exist in the game world simultaneously.

How many DIFFERENT types can be respawned at once.

How much loot should be respawned during restart to nominal the default is 0. Changing to 100 the server will attempt to respawn all loot to the nominal amounts in the types.xml unless the nominal is already present on the map.

How many initial tests are allowed for item spawn. Example: If an item has "SpawnInitial" set to 5, the server will try five times to spawn it in its designated location. If all five attempts fail, the item won't appear in that specific spot during that server session.

Penalty time for server hoppers, official only.

Default login time in seconds.

Default logout time in seconds.

Waiting time penalty for players who switch or disconnect from a server quickly.

Wetness and temperature on all items in the world. To turn off rotting food, change this AND FoodDecay to 0.

Maximum limit of spawned zombies across all zones in map. Increasing too much without reducing other events can cause lag.

Distance a player needs to be away for dynamic infected to spawn in nearby zone. The default value is typically 300 meters. This means that when a player is within 300 meters of a zone that does not currently have infected, the server will attempt to spawn infected in that zone to populate it and provide a challenge for players.
